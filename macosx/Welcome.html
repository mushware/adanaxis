Infernal Contractor II version 0.0.4-2

Take on the role of the software contractor and raise mayhem in this office simulation. This first stage sees you racing around the office against the clock in an attempt to convince people of your fitness.

Release notes

This software is a beta test. Please see the web site or installed README file for details. Please report any bugs or problems at the IC2 SourceForge project. To uninstall, delete the Infernal Contractor II directory

Known bugs

On machines without working 3D accelerators (like my G3/300MHz) the game will not work in fullscreen mode. If you have one of these, the game wil fade the screen three times whilst attempting to select a mode, and then give up. To get some limited operation, edit the configuration file start.txt to use windowed mode. The latest libsdl (http://www.libsdl.org) might resolve the problem - something after version 1.2.4.
